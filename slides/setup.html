1.1. Windows / Ubuntu

1.1.1. WSL

Install WSL on widows by running wsl --install in powershell. Read more setup instructions here

1.1.2. NodeJS

In WSL/ Ubuntu run the following to install nodejs (NodeSource) 

sudo apt update && sudo apt upgrade
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - &&\
sudo apt-get install -y nodejs

This will install Node.js v18.*.*, the LTS version of node

1.1.3. VSCode

Download VSCode here Install the WSL extension

1.2. MacOS

1.2.1. Homebrew

Homebrew is a package manager for macOS. The installation instructions can be found on the website or paste the following into the macOS terminal 

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

1.2.2. NodeJS

Run 

brew install node@18
